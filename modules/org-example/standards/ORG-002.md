# ORG-002: Internal APIs must use .internal.gov.uk domain

- Internal service-to-service API calls MUST use `.internal.gov.uk` domain, not raw IPs or localhost in config.
- Rationale: DNS-based service discovery enables load balancing, failover, and certificate validation
- Enforcement: Automated (regex for hardcoded IPs or non-.internal.gov.uk URLs in config files)
