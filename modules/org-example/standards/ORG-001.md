# ORG-001: Use approved HTTP client library

- MUST use `requests` or `httpx` for HTTP calls. `urllib` and `http.client` are not approved.
- Rationale: approved libraries have consistent error handling, retry logic, timeout defaults; urllib has footguns
- Enforcement: Automated (regex for `import urllib` or `from http.client` in .py files)
