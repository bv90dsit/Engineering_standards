# ORG-001: Use approved HTTP client library

## Standard

Python services MUST use `requests` or `httpx` for HTTP calls. `urllib` and `http.client` are not approved for production use.

## Rationale

- Approved libraries have consistent error handling, retry logic, and timeout defaults
- `urllib` has footguns (no timeout by default, verbose API, poor error messages)
- Consistent HTTP library across the org simplifies debugging and shared middleware

## What good looks like

- All HTTP calls use `requests.get()` / `httpx.Client()`
- Shared retry/timeout configuration via a wrapper or middleware
- No `import urllib` or `from http.client` in production code

```python
# GOOD: Using requests with timeout and retry
import requests
from requests.adapters import HTTPAdapter, Retry

session = requests.Session()
retries = Retry(total=3, backoff_factor=0.5)
session.mount("https://", HTTPAdapter(max_retries=retries))

response = session.get("https://api.internal.gov.uk/users", timeout=10)
```

```python
# GOOD: Using httpx (async)
import httpx

async with httpx.AsyncClient(timeout=10.0) as client:
    response = await client.get("https://api.internal.gov.uk/users")
```

## What bad looks like

```python
# BAD: urllib — no timeout by default, verbose, poor error messages
import urllib.request
response = urllib.request.urlopen("https://api.internal.gov.uk/users")

# BAD: http.client — low-level, no retry, no connection pooling
from http.client import HTTPSConnection
conn = HTTPSConnection("api.internal.gov.uk")
conn.request("GET", "/users")
```

## Enforcement

| Mechanism | What is checked | When |
|-----------|----------------|------|
| **Automated (CI/CD)** | Regex for `import urllib` or `from http.client` in .py files | Every PR |

**Primary enforcement: Automated.** The VS Code extension and CI checker both flag urllib/http.client imports.

## Quick reference

```python
import requests
response = requests.get("https://service.internal.gov.uk/api", timeout=10)

# Or async:
import httpx
async with httpx.AsyncClient(timeout=10.0) as client:
    response = await client.get("https://service.internal.gov.uk/api")
```
