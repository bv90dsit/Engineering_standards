# DAT-004: Open data published where possible

## Standard

Non-personal, non-sensitive data produced by government services SHOULD be published as open data. Data SHOULD use open standards (CSV, JSON, OData) and be listed on data.gov.uk.

## Rationale

- Public data belongs to the public
- Enables innovation and third-party service development
- Reduces FOI burden by proactively publishing
- Supports evidence-based policy making

## What good looks like

- Datasets published on data.gov.uk with a DCAT metadata record
- Machine-readable formats (CSV, JSON, OData) — prefer CSV for tabular data, JSON for hierarchical
- Documentation including field descriptions, units, and known caveats
- Regular update schedule defined and met (e.g. monthly refresh published by the 5th)
- Feedback mechanism for data consumers (e.g. GitHub issues repo or contact email in metadata)
- Clear licensing using the Open Government Licence v3.0

## Enforcement

| Mechanism | What is checked | When |
|-----------|----------------|------|
| **Periodic audit** | Open data opportunities identified and acted on; datasets listed on data.gov.uk | Service assessment |

**Primary enforcement: Periodic audit.** Service assessments check whether teams have considered open data opportunities and published non-sensitive data appropriately.

## Quick reference

**data.gov.uk publishing steps:**

1. Prepare data in CSV/JSON with consistent column headers
2. Create a DCAT metadata record (see below)
3. Sign in to data.gov.uk as a publishing organisation
4. Add dataset via "Add a dataset" form or CKAN API harvest
5. Set update frequency and verify the resource URL resolves
6. Monitor dataset feedback and maintain the update schedule

**Example DCAT metadata (JSON-LD):**

```json
{
  "@context": "https://www.w3.org/ns/dcat#",
  "@type": "Dataset",
  "title": "School performance tables 2025",
  "description": "Key stage 4 results by school, academic year 2024-25",
  "issued": "2026-03-01",
  "modified": "2026-06-15",
  "publisher": { "@type": "Organization", "name": "Department for Education" },
  "license": "http://www.nationalarchives.gov.uk/doc/open-government-licence/version/3/",
  "distribution": [
    {
      "@type": "Distribution",
      "mediaType": "text/csv",
      "downloadURL": "https://example.gov.uk/data/ks4-results-2025.csv"
    }
  ],
  "temporal": "2024-09-01/2025-08-31",
  "accrualPeriodicity": "annual"
}
```

**Open Government Licence v3.0 reference:**

Use this standard licence text (or link) in your dataset metadata:

> Contains public sector information licensed under the Open Government Licence v3.0.
> https://www.nationalarchives.gov.uk/doc/open-government-licence/version/3/
