# ENG-001: Open source by default

## Standard

All new source code MUST be made open unless there is a clear, documented reason not to (e.g. keys, security-sensitive algorithms, or unreleased policy).

## Rationale

- Promotes reuse across government, reducing duplication and cost
- Enables external scrutiny, improving code quality and security
- Supports the Technology Code of Practice point 3: "Be open and use open source"

## What good looks like

- Code is in a public repository from day one
- Repositories include a `LICENCE` file in the repository root (use MIT for code, OGL v3 for documentation/content)
- Sensitive configuration is separated from code (see ARC-002)
- A clear README explains what the service does and how to contribute
- `.gitignore` excludes secrets, build artefacts, and IDE files before first commit

## Exceptions

Document exceptions in an Architecture Decision Record (ADR). Valid reasons include:

- Code that would reveal security vulnerabilities if published before a fix
- Keys, credentials, or secrets (these should never be in source code regardless)
- Unreleased policy that is embargoed

## Enforcement

| Mechanism | What is checked | When |
|-----------|----------------|------|
| **Automated (CI/CD)** | LICENCE file exists in repository root | Every PR |
| **Periodic audit** | Repo visibility is public; exceptions have a documented ADR | Service assessment and quarterly reviews |

**Primary enforcement: Automated.** The compliance checker verifies a licence file is present. Repo visibility (public vs private) is audited at service assessment.

**OGL v3** (for documentation/content — use this header in the file):

```text
Crown Copyright (c) [year], licensed under the Open Government Licence v3.0.
https://www.nationalarchives.gov.uk/doc/open-government-licence/version/3/
```

**CI check for licence file** (GitHub Actions snippet):

```yaml
- name: Check LICENCE file exists
  run: |
    if [ ! -f LICENCE ] && [ ! -f LICENSE ]; then
      echo "::error::No LICENCE file found in repository root"
      exit 1
    fi
```

**Decision guide — which licence to choose:**

| Content type | Licence | File name |
|---|---|---|
| Source code | MIT | `LICENCE` |
| Documentation, datasets, content | OGL v3 | `LICENCE` |
| Mixed repo | MIT for code + OGL notice in docs folder | `LICENCE` + `docs/LICENCE` |
